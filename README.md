# Data_Analytics_Injection_Moulding_Process
This project analyzes injection molding data to find parameters affecting quality across various conditions. 
It details preprocessing, statistical analysis, and visualization to support a new machine monitoring system.

In this project, experimental data from an injection molding machine have been collected
under three different manufacturing conditions: Normal1, Condition1, and Condition4.
The normal condition serves as the baseline, while the other conditions reflect deviations
caused by adjustments in process parameters such as temperature and injection
pressure. These deviations can impact product quality, potentially leading to increased
defect rates or inefficiencies in production. The company aims to implement a parameter
monitoring system that will provide early warnings when the machine operates outside of
optimal conditions. However, identifying the key parameters responsible for product
variation is essential to develop this monitoring system. The problem lies in analyzing the
collected data to determine which parameters significantly impact product quality across
different manufacturing conditions.

This project will provide valuable insights into optimizing the injection molding process
through data-driven analysis, leading to more consistent product quality and efficient
manufacturing.
